# Reglas de Negocio

## Productos

- Un producto pertenece a un único proveedor.
- La combinación Referencia + Color + Curva + Proveedor debe ser única.
- El stock nunca puede ser negativo.
- El precio de venta es opcional.
- La foto del producto es opcional.

---

## Inventario

- El stock solo puede modificarse mediante movimientos.
- Todo movimiento debe quedar registrado.
- Existen tres tipos de movimientos:
    - Entrada
    - Salida
    - Devolución

---

## Clientes

- Una remisión siempre pertenece a un cliente.

---

## Remisiones

- Una remisión puede tener uno o muchos productos.
